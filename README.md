# Cardiovascular Disease Prediction Using Machine Learning

This project uses machine learning techniques to predict cardiovascular disease using patient medical data.

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest

## Techniques Applied

* Data Cleaning
* Exploratory Data Analysis
* Feature Engineering
* Hyperparameter Tuning
* Model Evaluation

## Deployment

A Streamlit web application was developed for interactive prediction.
